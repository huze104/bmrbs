# 介绍 #
基于flex，springside构建会议室预定系统
# 协议 #
系统是基于GPL协议开源，这里要求任何对于本系统修改或者基于本系统而衍生的系统，都要将源码开放出来，供其他人自由使用。
之所以，鉴于此，是因为一个会议室预定系统不牵涉商业机密，更难构成为产品核心竞争模块，但是对于oa类产品而言又是必不可少。那么本系统的根本使命就是能够提供一版好用的会议室预定系统。故而，若你改进了本系统，请考虑开源出来！
